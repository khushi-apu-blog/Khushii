Topic: Markdown, LaTeX, and Digital Documentation

This was also very new topic for me Markdown, LaTeX, and Digital Documentation .I never have used tools like Markdown, Zettlr, or Hugo.  I’ve only used the Google Docs Microsoft Word Notepad Wordpad for typing. So in the beginning , in the lab I felt a little lost but later It becomes so interesting for typing in zettler using markdown.

Typing in Markdown might be look simple initially but is not that easy. Any small mistakes can causes the whole output to be changed. ‘*’ and ‘#’ have a very significant and confusing role in markdown typing.

For me, personally, I found the most difficult part was LaTeX. Because writing mathematical expressions using code was completely new thing for me. In Lab also, when we had to type formulas I was confused that how to it then i took the help of my friends and also went through the website again and again. I tried multiple times to get that right. Also, i was very curious and doubtful so, I used to see the result immediately while typing., which was a bit frustrating.

At some point, I even felt like this is too technical or hard for me. But after practicing typing more and more in LaTex and later in labs, it slowly became easier for me. Now I started getting that how the syntax works and why it is useful.

When we were establishing and publishing the website using Hugo and GitHub Pages, I felt a small sense of achievement because it is a personalised website and also created by me. But I also struggle in this also . Even I am still not perfect at it, but now I feel more confident than before.

Overall, these were challenging but also interesting, especially because everything was new to me.
